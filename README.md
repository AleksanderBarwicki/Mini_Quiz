# Mini Quiz
 
### Opis projektu
Aplikacja mobilna na system Android służąca do rozwiązywania prostego quizu wiedzy ogólnej. Aplikacja losuje 5 pytań z bazy, zlicza punkty i pozwala na resetowanie postępu.
 
### Autor
Aleksander Barwicki
 
### Instrukcja uruchomienia
1. Pobierz kod źródłowy lub sklonuj repozytorium.
2. Otwórz projekt w programie **Android Studio**.
3. Poczekaj na synchronizację plików Gradle.
4. Uruchom aplikację na emulatorze (np. Pixel 5, API 33) lub urządzeniu fizycznym za pomocą przycisku "Run".
